LetterAI는 웹 인터페이스를 통해 OpenAI API를 더 쉽게 사용할 수 있도록 만든 도구입니다.
그러나 OpenAI API에 대한 비용은 별도로 청구되므로 사용자는 개인의 API 키를 입력해야 합니다.
OpenAI에서 결재방법(신용카드)을 등록해야지만 OpenAI의 키를 사용할 수 있습니다.
자세한 설명은 아래 문서를 참조해주세요.

API keys 발급

  • API-Key 생성 바로가기 ▷OpenAI

  • OpenAI 링크에 접속하여 로그인/회원가입을 진행합니다.







  • API Keys 생성 화면에서 +Create new secret key 버튼을 클릭합니다.

  • 진행하려는 프로젝트 또는 원하는 API 제목을 작성합니다.

  • 발급된 Key를 확인하고, 이때 보안을 위해 본인 Key값을 따로 저장해두는 것을 추천합니다.







결제 수단(신용카드)등록

  • 오른쪽 상단의 메뉴바에서 "Billing"->"Overview 화면을 선택합니다.

  • "Set up paid account"를 선택하고 개인 또는 회사의 지불 방법을 선택합니다.

  • 이후, 본인의 카드번호와 주소 등 정보를 입력합니다.

  • 결제 수단을 등록하고 나면 API Key가 사용가능합니다.
